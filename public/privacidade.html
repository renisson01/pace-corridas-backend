Política de Privacidade

Última atualização: 22 de março de 2026

A REGENI ("nós", "nosso") opera o aplicativo REGENI (o "Serviço"). Esta página informa sobre nossas políticas de coleta, uso e divulgação de dados pessoais.

1. Dados que Coletamos

  • Dados de cadastro: nome, email, senha (criptografada com bcrypt)
  • Dados de saúde: peso, altura, frequência cardíaca, sono, humor, exames de sangue, composição corporal — apenas quando você os fornece voluntariamente
  • Dados de treino: atividades GPS, distância, pace, elevação — sincronizados via Strava com sua autorização
  • Dados de pagamento: processados pelo Mercado Pago. Não armazenamos dados de cartão.

2. Como Usamos os Dados

  • Personalizar treinos e recomendações da IA Treinadora
  • Calcular idade biológica e scores de saúde
  • Exibir ranking e resultados de corridas
  • Melhorar o Serviço com base em dados agregados e anonimizados

3. Compartilhamento

Não vendemos seus dados pessoais. Compartilhamos dados apenas com:

  • Strava: via OAuth autorizado por você, apenas para sincronizar atividades
  • Mercado Pago: para processar pagamentos
  • Anthropic (Claude): para gerar respostas da IA Treinadora — sem identificação pessoal

4. Segurança

Senhas são criptografadas com bcrypt. Conexões usam HTTPS. Tokens JWT expiram periodicamente. Dados sensíveis de saúde são armazenados em banco PostgreSQL com acesso restrito.

5. Seus Direitos (LGPD)

  • Acessar seus dados a qualquer momento via o app
  • Corrigir dados incorretos no perfil
  • Solicitar exclusão total da conta enviando email para contato@regeni.com.br
  • Revogar autorização do Strava a qualquer momento

6. Retenção

Mantemos seus dados enquanto sua conta estiver ativa. Ao solicitar exclusão, removemos todos os dados pessoais em até 30 dias.

7. Cookies

Usamos localStorage para manter sua sessão (token JWT). Não usamos cookies de rastreamento de terceiros.

8. Menores

O Serviço é destinado a maiores de 16 anos. Não coletamos intencionalmente dados de menores de 16 anos.

9. Contato

Para questões sobre privacidade: contato@regeni.com.br
